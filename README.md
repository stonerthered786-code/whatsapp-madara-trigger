# WhatsApp Madara Trigger ⚡

Cinematic Tampermonkey script to open Circle App layout from selected ID in WhatsApp.

## Features
- Select ID → click Open
- Fullscreen animation (Madara style)
- Auto open layout page
- Smooth fade in/out

## Install

1. Install Tampermonkey
2. Open:
https://raw.githubusercontent.com/YOUR_USERNAME/whatsapp-madara-trigger/main/script.user.js
3. Click Install
